# PEDILO — Glosario de Siglas

> Referencia rápida de siglas usadas en toda la documentación del proyecto.
> Ordenado por temática.

---

## 🏗️ Producto y negocio

| Sigla | Significado |
|---|---|
| **MVP** | Minimum Viable Product — versión mínima funcional para validar |
| **SaaS** | Software as a Service — modelo de software por suscripción |
| **B2B / B2C** | Business-to-Business / Business-to-Consumer |
| **GTM** | Go-To-Market — estrategia de salida al mercado |
| **KYC** | Know Your Customer — verificación de identidad de cliente |
| **PII** | Personally Identifiable Information — datos personales identificables |
| **NFR** | Non-Functional Requirements — requisitos no funcionales (performance, seguridad, etc.) |
| **SLA** | Service Level Agreement — acuerdo de nivel de servicio |
| **RTO** | Recovery Time Objective — tiempo máximo aceptable para restaurar el servicio |
| **RPO** | Recovery Point Objective — máxima pérdida de datos aceptable medida en tiempo |
| **POS** | Point of Sale — punto de venta |
| **KDS** | Kitchen Display System — sistema de display de cocina |
